Description

Fight Together With Us is a top down strategy game that help in increase awareness of covid.

Player need to keep concentrate on what happening on the map and react by placing match unit to the area(Police, Medical or Grab).

This was a group work project where i code the game mechanic and all the gameflow in Unity
